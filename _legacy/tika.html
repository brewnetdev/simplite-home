Hierarchy

Goal → Story → Feature → Task
4단계 계층형 연결.

분기 목표(Goal)부터 오늘의 할 일(Task)까지 같은 구조 안에 놓입니다. 티켓 하나를 보면 상위 목표가, 목표를 보면 하위 진행 상황이 한눈에 들어옵니다.

Feature Set

작은 팀에게도 충분하고,
큰 팀에게도 부족하지 않은.

01

칸반 보드 + 드래그 앤 드롭

Backlog 사이드바와 TODO / In Progress / Done 3칼럼. 카드 이동·정렬이 즉각 반영되며 실시간 필터·검색도 한 자리에.

02

티켓 CRUD · 4가지 타입

Goal·Story·Feature·Task. 우선순위, 시작·마감일, 체크리스트, 라벨, 댓글, 담당자 배정까지 티켓 단위에 모두 담깁니다.

03

워크스페이스 & 권한

OWNER / MEMBER / VIEWER 3단 역할. 24시간 만료 초대 링크로 외부 협업도 안전하게.

04

WBS · 간트 차트

일정 기반 시각화로 목표 대비 현재 진행률을 한눈에. 팀 대시보드에는 번다운과 워크로드도 함께.

05

Slack · Telegram 알림

채널 연동과 마감 D-1 자동 알림. 헤더 벨 드롭다운으로 앱 내에서도 놓치지 않습니다.

06

검색 · 고급 필터

실시간 검색과 필터 칩. 라벨·우선순위·날짜 범위를 조합한 뷰를 저장해 팀별로 공유하세요.

07

Google OAuth · 온보딩

NextAuth.js v5 기반 안전한 로그인. 첫 진입 시 워크스페이스 선택·생성 흐름까지 자동으로.

08

MCP 네이티브

모든 티켓·워크스페이스가 MCP로 노출. Claude Desktop, Cursor 등 외부 에이전트가 Tika 데이터에 직접 접근합니다.

09

AI 자동화 PRO

회의록에서 티켓 자동 생성, 우선순위 제안, 번다운 이상 감지. 팀 플랜에서 점진적으로 제공됩니다.
